OrangeAudio

Ever Wished you could visualize live audio and identify the Note Played from an input source ?

Presenting to you Orange Audio a simple and elegant audio visualizer. Custom config file allows you to change color size density and other parameters As an added bonus based on the amplitude(s) of the different frequencies being played the Note Played is also shown. Works best with A Grand Piano / Keyboard .Get Hacking !

Instructions to install all dependencies for realtime note detection 

OpenGL 	- sudo apt-get install freeglut3-dev
ALSA 	- sudo apt-get install alsa_utils
	  sudo apt-get install alsamixer
fftw3 	- sudo apt-get install fftw3-dev
Audio	- sudo apt-get install libasoun2-dev

compile - g++ scratch.cpp -lGL -lGLU -lglut -lfftw3 -lasound2
